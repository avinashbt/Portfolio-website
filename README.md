## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/avinashbt/Portfolio-website.git
```

Navigate to project folder:

```bash
cd Portfolio-website
```

Open in browser:

```bash
index.html
```

Or if using React:

```bash
npm install
npm start
```
